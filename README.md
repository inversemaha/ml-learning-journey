# 🚀 My ML Learning Journey

This repository outlines my beginner-level Machine Learning learning plan. As a backend Laravel developer, I am learning ML to keep pace with evolving technology trends. This README provides instructions for setting up a clean environment, understanding essential tools, and accessing curated resources for efficient hands-on learning.

---

## 1️⃣ Create a Virtual Environment

Creating a **virtual environment** helps isolate your project’s dependencies from your system Python.

```bash
python3 -m venv venv
```

---

## 2️⃣ Activate the Virtual Environment

Activate the environment to start using it:

```bash
source venv/bin/activate
```

You should see:

```bash
(venv) maha@pc:~/ml-learning$
```

> This `(venv)` prefix means your environment is active.

---

## 3️⃣ Upgrade `pip`

Upgrade `pip` (Python package manager) to ensure smooth installation of packages:

```bash
pip install --upgrade pip
```

---

## 4️⃣ Install Required Packages

```bash
pip install pandas scikit-learn matplotlib jupyter
```

Here’s what each package does with **resources for faster learning**:

### 📊 **pandas**
[Pandas](https://pandas.pydata.org/) is a library for **data manipulation and analysis**.  
- Load datasets (CSV, Excel)  
- Clean and preprocess data  
- Explore and analyze data  
- **Learn faster:** [Pandas Tutorial](https://pandas.pydata.org/docs/getting_started/intro_tutorials/index.html)

### 🤖 **scikit-learn**
[Scikit-learn](https://scikit-learn.org/) is a **Machine Learning library**.  
- Build ML models like Linear Regression, Decision Trees, SVM  
- Split data into training and testing sets  
- Evaluate model performance  
- **Learn faster:** [Scikit-learn Tutorials](https://scikit-learn.org/stable/tutorial/index.html)

### 📉 **matplotlib**
[Matplotlib](https://matplotlib.org/) is a **plotting library**.  
- Visualize data (line charts, bar charts, histograms)  
- Check distributions and trends  
- **Learn faster:** [Matplotlib Tutorial](https://matplotlib.org/stable/tutorials/introductory/pyplot.html)

### 📓 **jupyter**
[Jupyter Notebook](https://jupyter.org/) allows you to **write and run code in an interactive notebook**.  
- Combine code, visualizations, and notes  
- Experiment with ML models interactively  
- **Learn faster:** [Jupyter Notebook Guide](https://jupyter-notebook.readthedocs.io/en/stable/notebook.html)

---

## 5️⃣ Start Jupyter Notebook

```bash
jupyter notebook
```

> This will open a browser window where you can create `.ipynb` files and start coding your ML experiments.

---

## 6️⃣ Deactivate the Virtual Environment

When done, deactivate the environment:

```bash
deactivate
```

---

## 💡 Tips
- Always **activate your virtual environment** before running code.  
- Use **small datasets** while learning to save computation time.  
- Document experiments and outputs inside **Jupyter notebooks**.
